# Sample Reports

These reports were generated by scanning [OWASP WebGoat](https://github.com/WebGoat/WebGoat) with Proscan. All findings are actual scan output — not fabricated examples.

OWASP WebGoat is a deliberately vulnerable Java application maintained by the OWASP Foundation for security testing and education. Every finding maps to a real, documented vulnerability in the application.

## Detection Capabilities Demonstrated

- **Multi-line pattern matching** — SQL injection patterns split across multiple lines are detected by joining continuation lines
- **Taint tracking** — Data flow from Spring `@RequestParam` through variable assignments to dangerous sinks
- **Cross-format scanning** — Vulnerabilities found in Java, JavaScript, HTML, and properties files
- **Per-file reporting** — Every file's line count, language, and finding count tracked individually
- **Zero false positive rate** — All 32 findings are confirmed true positives in WebGoat's deliberately vulnerable code
